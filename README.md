# Aluguel Veículos

## Esse projeto é um desafio proposto por IA.
 O projeto consiste em uma ferramenta de suporte para um sistema de alugueis de veículos, onde é possível cadastrar os veículos disponíveis, os clientes do estabelecimento, verificar disponibilidade, calcular o valor de acordo com os dias de locação do veículo, conseguir alugar e devolver e sistema para aplicação de descontos.
Construído com estrutura Java Padrão, utilizando noções de Programação Orientada a Objetos, Clean Code, Lógica de programação, Testes Unitários com JUnit.

## Após instalar o projeto basta ter o compilador do Java (JDK 24) e importar as bibliotecas do JUnit.

## Instrução de uso
1. Para rodar a aplicação, entre em SistemaAluguel (classe main) e compile.
2. Para utilizar os Testes Unitários, entre em TestSuite (tester de todos os testes) e compile.
